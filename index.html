    Introduction

    Welcome to the Succession Link API.

    Authentication

    Authentication is done via an API key. Include an "Authorization" header in each request. The format is as follows:

    Authorization: Token YOUR_TOKEN

    Valuations

    Quick Valuation

    The quick valuation provides a simple valuation based on a minimal number of attributes. 

    import json
import requests
data = {
    "revenue": {
        "year": 2023,
        "aum": 1000000000,
        "revenue": 10000000,
        "expenses": 0,
        "interest_expense": 0,
        "noncash_expenses": 0,
        "office_comp": 0,

    },
    "business_info": {
        "avg_client_age" : "50",
        "business_name" : "Tims LLC",
        "city" : "Vista",
        "num_clients" : "50",
        "num_employees" : "50",
        "ownership_percentage" : 100,
        "percent_commission_based" : 10,
        "percent_fee_based" : 90,
        "state" : "California",
        "street1" : "123 Main Street",
        "street2" : "",
        "zip" : "92084"
    }
}

url = "https://successionlink.com/api/v1/valuation/quick/"
r = requests.post(
  url,
  headers={
    'content-type': 'application/json',
    'authorization': 'Token YOUTOKEN'
  }, data=json.dumps(data))
print(json.dumps(r.json(), indent=2))
